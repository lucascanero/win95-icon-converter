# Win95 Icon Converter

Transform any image into a retro Windows 95–style icon, instantly, right in your browser!  
Live, private, and blazing fast. Powered by pure HTML5, CSS3, and JavaScript.

- **Image to Win95 Icon** – Give any image that nostalgic pixel-art feel
- **Custom RGB Background** – Choose your perfect backdrop color
- **Instant Preview** – See changes live as you adjust
- **Classic Pixelation** – Pixel size and color palette controls
- **Outline Shadow** – Add an authentic retro shadow border
- **Multiple Languages** – English, Portuguese, Spanish

## 🛠️ How to Use

1. **Upload an Image**  
   PNG, JPG, WebP, or SVG – drag & drop or click "Upload image".
2. **Customize**  
   - Choose background color (or transparent).
   - Pick final icon size (32/64/128/256 px).
   - Adjust pixelation and color palette.
   - Enable outline shadow, if you want.
3. **Preview**  
   Instantly see original vs Win95-style version.
4. **Download**  
   Click “Download PNG” to save your retro icon.
